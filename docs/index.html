QOwnNotes Scripting Wiki

Notes and documentation for various QOwnNotes scripts in development.

  • Project repo (GitHub)

This wiki is also a live demonstration of the export-notes-as-website and backlinks scripts, and is entirely written in and created using QOwnNotes. This is proof that QOwnNotes can be an excellent CMS and quick static site generator for building simple websites.

Topics

The following main topics (or groups) are covered in this wiki:

  • Scripts - Documentation for scripts under development
  • Issues - Bugs, feature requests, quirks etc
  • Examples - Examples of various features to help with duplicating and resolving issues
  • Diagrams - Some sample diagrams for testing Mermaid integration
  • Utilities - Command-line utility scripts for working with linked notes
